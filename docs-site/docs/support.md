# 프로젝트 후원하기

Home-Assistant-Matter-Hub는 제가 여가 시간에 유지보수하는 무료 오픈소스 커뮤니티 포크입니다. 후원은 전적으로 자발적이며, 이 프로젝트는 앞으로도 항상 무료입니다. 이 프로젝트가 여러분의 시간을 절약해 주거나 스마트 홈을 원하는 대로 작동하게 해준다면, 작은 후원이 동기 부여를 높이고 테스트를 위한 호스팅 및 하드웨어 비용을 충당하는 데 도움이 됩니다.

## 후원 방법

### GitHub Sponsors

GitHub를 통한 정기 또는 일회성 후원입니다. 제 GitHub 프로필에 표시되며 카드 또는 PayPal로 결제할 수 있습니다.

- [github.com/sponsors/RiDDiX](https://github.com/sponsors/RiDDiX)

### Ko-fi

일회성 또는 월별 후원입니다. Ko-fi 계정이 필요 없으며, 카드 정보를 입력하거나 PayPal을 사용하면 됩니다. 해외에서도 편리하게 이용할 수 있습니다.

- [ko-fi.com/riddix](https://ko-fi.com/riddix)

### PayPal

일회성 후원을 위한 PayPal 직접 링크입니다. 이미 PayPal 계정이 있다면 전 세계 어디서나 사용할 수 있습니다.

- [paypal.me/RiDDiX93](https://www.paypal.me/RiDDiX93)

## 후원자 목록

현재 및 과거 후원자 목록은 [README](https://github.com/RiDDiX/home-assistant-matter-hub#-support-the-project)에서 확인할 수 있습니다. 후원해 주신 모든 분들께 감사드립니다. 여러분의 후원은 정말 큰 힘이 됩니다.
